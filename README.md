# Overview

Shooting method to solve the flow instability of Poiseuille flow.

# Installation

1. install dislin
  1) `tar -xf dislin-11.3.linux.i586_64.tar.gz`
  2) `cd dislin-11.3`
  3) `export DISLIN=$HOME/dislin`
  4) `./INSTALL`
2. compile the main code
  1) `cd ../src_f90`
  2) `make`
  3) `make run`
3. View the result: `evince dislin.eps`
